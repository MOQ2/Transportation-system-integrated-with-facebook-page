Getting Your Page ID
  1. Go to your Facebook page
  2. Click on "About" in the left sidebar
  3. Scroll down to find your Page ID
  4. Copy the ID and paste it here
Getting an Access Token
  1. Go to Facebook for Developers
  2. Create a new app or use an existing one
  3. Add the "Facebook Login" product
  4. Go to "Graph API Explorer"
  5. Select your app and page
  6. Request permissions: pages_read_engagement, pages_manage_posts
  7. Generate a token and paste it here
Page access tokens typically expire after 60 days. For long-term use, you'll need to generate a long-lived token.
